Mininet: Rapid Prototyping for Software-Defined Networks

Mininet is the easy way to do cool stuff with OpenFlow and Software-Defined Networks. Create a complete virtual network on a single machine (VM, cloud, or native) in seconds with a single command:
You can interact with your network using the Mininet CLI (and API), customize it, share it with others, or deploy it on real hardware.
Actively developed and supported, Mininet is useful for interactive development, testing, and demos, running any OpenFlow controller or virtual switch. Permissive BSD license.
